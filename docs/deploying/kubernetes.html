Deploy to the Cloud with Kubernetes

Prerequisites

Whilst Kitura can be deployed to any Cloud offerings we will be using IBM Cloud as an example in this guide.

To follow this guide you will need to have completed the following:

  1. Create IBM Cloud account
  2. Created a free cluster on IBM Cloud
  3. Created/already have a Cloud ready Kitura application
  4. Completed the Docker guide
