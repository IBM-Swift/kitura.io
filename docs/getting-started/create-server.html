Swift Package Manager

The Swift Package Manager (SPM) is a tool for managing the distribution of Swift code, similiar to npm for Javascript. Kitura can be added to a Swift project using SPM and this guide will take you through how to do that as well as creating an Xcode project if you're working on macOS. SPM is supported on both Linux and macOS.

Kitura CLI

The Kitura CLI is an open source command-line interface for Kitura. The Kitura CLI provides numerous options for creating a Kitura server depending on your requirements, ranging from basic servers to cloud ready deployments via Kubernetes. Kitura CLI is supported on both Linux and macOS.

Kitura macOS App

The Kitura macOS app provides a UI version of the Kitura CLI, suitable for those wishing to avoid the terminal. The Kitura macOS app provides fewer choices for project creation than the CLI but does provide an easy to use UI experience, and offers the 3 most common choices for project creation. The macOS app is available on the latest version of macOS.
