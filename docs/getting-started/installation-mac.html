Installation for macOS

Step 1: Install Xcode

Download and install the latest version of Xcode from the App Store.

To confirm whether Swift has been installed successfully open a terminal and run the following:

swift --version

You should see something similar like this:


Apple Swift version 4.1.1 (swift-4.1.1-RELEASE)
Target: x86_64-apple-darwin18.0.0

If you got an error similar to this:

swift: command not found

Then you may need to manually install the Xcode command line tools.

You can do that by typing the following into a terminal:

xcode-select --install

You can also do this via Xcode:

  1. Open Xcode preferences using the shortcut: CMD + ,
  2. Click on the Locations tab
  3. From the Command Line Tools drop down ensure the correct version of Xcode is selected.

Optional: Install Kitura CLI

Install via brew

For this installation you will need to have Homebrew installed.

If you do not have Homebrew installed you can follow the Homebrew installation steps

Then run the following in a terminal:

brew tap ibm-swift/kitura
brew install kitura

Install via npm

For this installation method you need to have npm installed.

If you do not have npm installed you can follow the npm installation steps.

Then run the following in a terminal:

npm install -g kitura-cli

Verify the Kitura CLI is installed by running the following:

kitura --version

You should see output similar to this:

0.0.12
