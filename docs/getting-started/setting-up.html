Setting up a workspace

Setup: macOS

Download and install the latest version of Xcode.

Install the command line tools for Xcode

xcode-select --install

Setup: Ubuntu Linux

Kitura is tested on Ubuntu 14.04 (LTS) and Ubuntu 16.04 (LTS).

Swift development on Ubuntu Linux requires various Linux system packages.

sudo apt-get update
sudo apt-get install clang libicu-dev libcurl4-openssl-dev libssl-dev libpython2.7
