Step 2: Initilise HeliumLogger

HeliumLogger can be customised to change the level of logging.

This means you can decide how much information you'd like to be logged, based on importance.

If you want to use HeliumLogger out of the box then you only need the following:

HeliumLogger.use()

If you'd like to customise the level of logging then first you need to create an instance of HeliumLogger:

let logger = HeliumLogger()

Next you need to set it as the Logger used by LoggerAPI:

Log.logger = logger

Step 3: Set logging level (Optional)

Logging levels:

  1. 1. entry
  2. 2. exit
  3. 3. debug
  4. 4. verbose (default)
  5. 5. info
  6. 6. warning
  7. 7. error

Verbose is set by default. Setting a particular logging level also includes all levels above it.

For example, if you set a logging level of `verbose`:

HeliumLogger.use(.verbose)

You will get logging for; `verbose`, `info`, `warning` and `error`.

Similarly if you set a logging level of `warning`:

HeliumLogger.use(.verbose)

You will get logging for `warning` and `error`.
