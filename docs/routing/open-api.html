Kitura OpenAPI

Prerequisites

  • Kitura Server: Learn how to create one in our Getting Started guide
  • Codable Routes: Learn how to add Codable routes in our Codable Routing guide

Step 1: Add Kitura OpenAPI to your project

In your Package.swift you need to add the following:

.package(url: "https://github.com/IBM-Swift/Kitura-OpenAPI.git", from: "x.x.x"),

Where x.x.x represents the latest version

For more information on adding Kitura OpenAPI to your project you can view the Readme.

Step 2: Build your project

Since you've added a new package to your Package.swift you need to rebuild your project.

In a terminal run the following:

swift package update

If you're on macOS and using Xcode you will also need to generate a new xcode project:

swift package generate-xcodeproj

Step 3: Add Kitura OpenAPI to your App code

Import Kitura OpenAPI into your project

import KituraOpenAPI

Add Kitura OpenAPI to your router:

KituraOpenAPI.addEndpoints(to: router)

Step 4: Test your RestAPIs

Start your Kitura server
